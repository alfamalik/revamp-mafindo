WHB (Whatsapp Hoax Buster)

Untuk membantu sukarelawan kami memerangi tipuan di WhatsApp, kami telah membuat WHB (WhatsApp Hoax Buster). Saat ini (versi 1) adalah ekstensi Google Chrome, dengan kemampuan berikut:

  1. Automated Counter-Hoax: Tidak perlu campur tangan manusia.

  2. Tancapkan ke mesin Yudistira kami: masukkan kunci API Yudistira Anda, untuk mulai menggunakan WHB. Mesin ke mesin: tidak ada solusi sensor.

  3. Mesin ke mesin: tidak ada solusi sensor.

  4. Peluang pendidikan publik: dengan melakukan kontra-tipuan, alih-alih menyensor, kami melakukan upaya pendidikan publik.

  5. Deteksi dini: dengan memungkinkan orang untuk melaporkan ke instalasi WHB kami, ini pada gilirannya akan memungkinkan kami untuk dengan cepat mendeteksi tipuan baru sebelum menjadi viral - dan melawannya.

Penggunaannya mudah, cukup unduh & instal ini di browser Chrome Anda, lalu masukkan kunci API Yudistira Anda:
